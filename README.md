## 📦 Run Locally

```bash
git clone https://github.com/Dearabhay/abhaykumar.git
cd abhaykumar
npm install
npm run dev
```

## 🚀 Deploy on Vercel

1. Upload this folder to GitHub
2. Open [vercel.com](https://vercel.com) → Add New Project
3. Import the GitHub repository
4. Click Deploy
